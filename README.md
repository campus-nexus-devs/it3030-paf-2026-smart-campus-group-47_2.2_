# 🏫 Smart Campus Operations Hub - Campus Nexus

## 📌 Project Overview

Campus Nexus is a web application for university campus operations management including facility bookings and maintenance ticketing.

### Technology Stack

| Layer | Technology |
|-------|------------|
| Backend | Spring Boot 3.2.4 |
| Frontend | React.js 18.2.0 |
| Database | MySQL 8.0 |
| Authentication | OAuth2 (Google Login) + JWT |
| Build Tool | Maven |
| API Testing | Postman |

### User Roles

| Role | Permissions |
|------|-------------|
| **USER** | Book resources, create tickets, view own bookings |
| **ADMIN** | Full access, approve bookings, manage resources |
| **TECHNICIAN** | Handle assigned tickets, update status |
